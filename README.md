# Bus-booking-System

## Overview

A cloud-based online bus reservation system where users can search buses, book seats, and manage reservations and get email confirmation after a successful booking

## Features

- Search buses
- Seat booking
- Booking confirmation
- Passenger details
- Booking history
- Email Notification after booking successful

## Technologies

- HTML
- CSS
- Flask
- Python

## AWS Services

- EC2
- Lambda
- API Gateway
- DynamoDB
- SNS

## Architecture

User

↓

Website

↓

API Gateway

↓

Lambda

↓

DynamoDB

|

SNS
